# Miam Miam le chocolat
TP noté final du cours de conception logiciel

### Partie webservice : 
Depuis la racine : <br>
``` $ cd Webservice ``` <br>
``` $ pip install -r requirements.txt ``` <br>
``` $ uvicorn main:app --reload ``` <br>
Sur le navigateur on va là http://127.0.0.1:8000/docs#/ <br>

### Partie client: 
Depuis la racine : <br>
``` $ cd Client ``` <br>
``` $ pip install -r requirements.txt ``` <br>
``` $ python3 main.py ``` <br>
ou alors <br>
``` $ python main.py ``` <br>
